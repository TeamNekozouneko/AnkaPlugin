<h1 align="center">AnkaPlugin</h1>

<p align="center">チャット欄が某掲示板仕様になるネタプラグイン</p>

## 概要

* ただのネタプラグインですのでコードの雑さには目をつぶっていただけると幸いです。

## ドキュメント

* チャットに>>1などアンカーを送信してホバーしてみてください。
* /aku playerでアク禁/解除できます。
* /clsでチャットを初期化します。
